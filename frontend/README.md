## 📝 Minimal Requirements

- NodeJs 12.x
- Npm 6.14.x

## 🚀 Install

```sh
$ https://github.com/MaduFernandes/library-PWI.git
```

## 📝 Configure

```sh
$ cp .env.example .env
```

```sh
$ npm install
```

Para iniciar o projeto:

```sh
npm start
```
